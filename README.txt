Frogger Game:

Bugs travel left to right on random pavement tracks at random speeds. 3 on screen at any one time.
Player starts at center of grass at bottom
Player moves up down, right, left, one block at a time.
If player collides with a bug, he is reset to initial position and the game number increments.
If player makes it to the water, game number increments, wins increment, 
he basks in his glory for 1 second, then gets reset to the start position.
Player cannot move off screen.
Refresh will re-initialize the game and the counters.
